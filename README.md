# MarketSearch

這是一個用於股票、ETF、基金應用程式，透過簡單搜索與簡潔頁面讓用戶快速取得資訊

## 目錄

- [功能](#功能)
- [安裝](#安裝)


## 功能

- 1.股票搜尋
- 2.ETF搜尋
- 3.基金搜尋


## 安裝

### 1. 克隆專案

```bash
git clone https://github.com/MarkLo127/marketsearch.git
```

### 2. 進入專案目錄
```bash
cd marketsearch
```
### 3. 安裝所需的依賴
```bash
pip install -r requirements.txt
```
### 4. 啟動應用程式
```bash
streamlit run search.py
```
